## React Weather App

University project

## Available Scripts

In the project directory, you can run:

### `npm install`
### `npm start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.


## Technology stack

- TypeScript
- SCSS
- React + hooks
- Redux Toolkit
- Redux Thunk
- Axios
- Nivo

## API
